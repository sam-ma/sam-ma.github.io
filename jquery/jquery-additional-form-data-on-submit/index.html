jQuery Additional Form Data on Submit
Last modified at:   June 26, 2018                  jquery   

Problem:

Suppose you would like to use jQuery to submit a form with additional data.

Solution:

var input = $("<input>")
               .attr("type", "hidden")
               .attr("name", "myname").val("myvalue")
               .appendTo("#myform");

Replace

  1. myname to the parameter name you want to associate with the value.
  2. myvalue to the value you want to submit
  3. myform to the id of the form you would like to submit
